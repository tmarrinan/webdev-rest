Purpose

This website was created to show crimes that have taken place in Saint Paul, Minnesota. It breaks down St. Paul into it's 17 neighborhoods, and breaking the crimes down into 12 different categories. The user can see the data on the map and also a table showing the data corresponding to that data point.

How to Use

The user of this site can look at the data from a wide variety of Saint Paul neighborhoods. From the start, you'll have to enter in a URL for the server that serves the data about Saint Paul crime, and you're off! After the URL has been entered and the data has been pulled, you will see markers on the map that show the amount of incidents that took place within that neighborhood. Below you can then take a look at those crimes, along with the data recorded with it. Off to the right hand side you will be able to select what codes you would like to see along with the neighborhoods you want the data from. You also will be able to filter the results based on the start date and end dates, allowing you to look for crimes in a particular time frame. The user can also search up specific neighborhoods and get the data back from those ones. Lastly, under the filters the user can submit a new incident form where they will be prompted to enter all the data about that incident, then hit submit to enter it into the system.

Tools

This website was created using Foundation.css and custom css for the styling of the page which allowed us to create a fluid grid that makes it so the page is both stylied and continious across all platforms making it great to use on the go! We also use Vue javascript framework to which enabled us to build a wide variety of elements that we can pull and use many times over across the pages, allowing us to spend more time building the site to have more mechanics. We used Leaflet for the map and plotting the points onto that map. However, we used Nominatim for the locations, we can use their API to search for the longitude and latitude of the addresses for each incident and neighborhood. This allowed us to accurately and effeciently place the markers on the map.
